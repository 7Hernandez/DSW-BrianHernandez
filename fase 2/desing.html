Proposito del sitio web

Las etiquetas son la estructura básica del HTML. Estas etiquetas o tags se componen y contienen otras propiedades, como son los atributos y el contenido.

Etiquetas de HTML

¿Que es una etiqueta HTML?

El texto en una pagina web esta ´marcado´ con estos codigos para dar instrucciones al navegador web sobre como mostrar el texto. Estas etiquetas de marcado son las propias etiquetas HTML Cuando escribes codigo HTML, estas escribiendo etiquetas html.

  • El caracter "menor que" <
  • Una palabra o caracter que determina que etiqueta se esta escribiendo.
  • Cualquier numero de atributos HTML que se quiera usar, escritos de la forma nombre="valor"
  • El caracter "mayor que" >

Etiquetas HTML basicas

Hay una serie de etiquetas que son las mas usadas para caracter Cualquier documento HTML, a continuaion las explicamos.

  • Body - para el contenido
  • head -para informacion sobre el documento
  • div - division dentro del contenido
  • a - para enlaces
  • strong - para poner el texto en negrita
  • br - para saltos de linea
  • H1..H6 - para titulos dentro del documento
  • img - para añadir imagene al documento
  • ol - para listas ordenadas
  • ul - listas ordenadas
  • li - para elementos dentro de la lista
  • p - para paragrafos
  • span - para estilos de una parte del texto

ATRIBUTOS

Los atributos en HTML se pueden considerar como informacion adicional que se coloca siempre en la etiqueta de apertura , esto para añadir, complementar informacion o significado al elemento HTML al que estamos colocando

Un atributo se escribe de la sigueinte manera: atributo="valor"

    Atributo valores

    Id - su valor puede ser cualquier palabra que sirve para identificar un elemento
